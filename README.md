<br><br>
![Showcase image](.github/image.png)

## Controls
|     Button     |      Action      |
|----------------|------------------|
| Mouse Button 1 |  Place down sand |
| 1              |  Switch to air   |
| 2              |  Switch to sand  |
| 3              |  Switch to mud   |
| 4              |  Switch to water |
| F5             |  Reload shader   |
